# Termini Processuali vs Termini di Impugnazione — Analisi Strategica

Data: 2026-04-01

## Relazione tra i due concetti

**Termini di impugnazione è un sottoinsieme di termini processuali.**

I termini processuali coprono TUTTI i termini previsti dal codice di procedura (civile, penale, amministrativo, tributario). I termini di impugnazione riguardano esclusivamente le scadenze per impugnare provvedimenti giurisdizionali.

## Implicazioni strategiche per avvocatotools.it

### 1. Due pagine separate, non una sola
L'intent di ricerca è diverso:
- **"calcolo termini processuali"** → avvocato che deve calcolare qualsiasi scadenza procedurale
- **"calcolo termini impugnazione"** → avvocato che deve specificamente sapere entro quando impugnare

### 2. Priorità: impugnazione prima, processuali dopo
- La pagina impugnazione è più circoscritta e realizzabile subito
- La pagina processuali richiede un database completo di tutti gli articoli CPC con i relativi termini — progetto più grande
- Le due pagine si cross-linkano a vicenda (valore SEO)

### 3. Differenziazione vs Andreani
**Andreani (impugnazioni):** calcolatore puro + contenuto editoriale minimo (solo termini base per giurisdizione)
**Noi (impugnazioni):** calcolatore + contenuto editoriale profondo che include:
- Cluster bonus (licenziamento, testamento, delibera condominiale, cartella esattoriale)
- Riforma Cartabia e impatto sui termini
- FAQ basate su PAA (28 domande raccolte)
- Tabelle comparative complete con normativa linkata
- Casi particolari (sospensione feriale, sabato, PEC dopo le 21:00)

### 4. Calcolatore impugnazione — design
Input:
1. Tipo procedimento: Civile / Amministrativo / Tributario / Lavoro
2. Tipo impugnazione: Appello / Cassazione / Revocazione / Opposizione di Terzo
3. Tipo data: Notifica sentenza / Pubblicazione sentenza / Scoperta vizi
4. Data di decorrenza

Output:
- Data scadenza termine
- Giorni rimanenti
- Eventuale sospensione feriale applicata
- Articolo di riferimento con link a Normattiva
- Warning se scade di sabato/festivo (con data prorogata)
