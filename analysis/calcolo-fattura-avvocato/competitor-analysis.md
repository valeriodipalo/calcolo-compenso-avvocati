## 3. Key Findings

### 3.1 Universal gaps across ALL competitors

1. **No comprehensive editorial content**: The best competitor (StudioCataldi) has ~700 words of generic text. Nobody combines a full calculator with a 2000+ word guide. This is our primary differentiator.

2. **No FAQ section**: Zero competitors have an FAQ. Given the 15 PAA questions identified in keyword research, this is a massive SEO opportunity.

3. **No worked examples**: Nobody shows step-by-step calculation examples (e.g., "Avvocato in regime ordinario fattura per EUR 3.000 di competenze" with full breakdown). This is critical for both SEO and user comprehension.

4. **No schema markup** (except Andreani's AggregateRating): Nobody uses WebApplication, Article, or BreadcrumbList structured data.

5. **No acconto/saldo support**: No competitor handles the common scenario of fattura per acconto followed by fattura a saldo.

6. **No comprehensive normativa section**: References are scattered and incomplete. Nobody provides a full table of relevant norms with links.

### 3.2 Feature leader by category

| Category | Leader | Our approach |
|----------|--------|-------------|
| Calculator features | MioLegale (3 modes, all toggles) | Match + exceed (add acconto/saldo, IVA per cassa) |
| Editorial content | StudioCataldi (~700w) | 3-5x more content (2000-4000 words) |
| Brand recognition | Andreani (2900 branded SV) | Compete on content quality + SEO |
| UX / accessibility | None stands out | Mobile-first, no registration, fast loading |
| Schema markup | Andreani (AggregateRating) | WebApplication + Article + BreadcrumbList |

### 3.3 Differentiation opportunities

1. **Content + tool combo**: Be the ONLY page that combines a full interactive calculator with comprehensive editorial content (2000-4000 words).
2. **Calcolo inverso on same page**: Only MioLegale does this. Andreani sends users to a different page. We match MioLegale's 3-mode approach.
3. **Worked examples for every regime**: Regime ordinario, forfettario, split payment PA -- step-by-step breakdowns that answer PAA questions and target long-tail keywords.
4. **Acconto/saldo mode**: Entirely unaddressed by competitors. Targets "acconto fattura avvocato" (880 SV) and "fattura avvocato acconto e saldo" (10 SV).
5. **No registration wall**: Andreani gates the full calculator behind registration. We offer everything free.
