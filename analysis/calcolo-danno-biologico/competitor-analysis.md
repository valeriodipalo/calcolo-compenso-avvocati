# Competitor Analysis: Calcolo Danno Biologico

### 1. AvvocatoAndreani.it (Rank #1)
- **URL**: calcolo_danno_biologico.php
- **Type**: Calculator-focused (micropermanenti only)
- **Content**: Minimal editorial (~500 words), mostly form + legal article text
- **Calculator**: Micropermanenti only (1-9 punti), with ITT/ITP, danno morale %
- **Ratings**: 4.6/5 (2,355 reviews) — huge trust signal

- **Strengths**: Good editorial depth, practical focus on malasanità

### 4. Boccadutri.com (Rank #7 in secondary SERP)

- **FAQ**: 9 items
- **Strengths**: Most comprehensive content, covers TUN, good structure
- **Weaknesses**: No schema markup, commercial tone (law firm marketing)

### 5. OsservatorioAmianto.it (Rank #7)
- **URL**: danno-biologico
- **Type**: Editorial guide (no calculator)
- **Content**: ~2,500 words, amianto-focused angle
- **Strengths**: Good coverage of INAIL, tabelle Milano
- **Weaknesses**: No calculator, niche amianto focus, no TUN

## Key Gaps to Exploit
1. **No competitor has a single unified calculator** covering micro + macro + TUN
2. **No competitor has proper schema markup** (Breadcrumb, Article, WebApplication)
3. **No competitor has NormativaQuote components** with direct Normattiva links
4. **Most calculators use outdated values** (Milano 2021, DM 2022)
5. **No competitor covers the full topic** in a single comprehensive page

## Recommended Page Structure
1. Calcolatore (unified: micropermanenti + macropermanenti + TUN)
2. Cos'è il Danno Biologico (definizione, inquadramento giuridico)
3. Micropermanenti (art. 139 CdA, formula, tabella ministeriale)
4. Macropermanenti (tabelle Milano 2024, come funzionano)
