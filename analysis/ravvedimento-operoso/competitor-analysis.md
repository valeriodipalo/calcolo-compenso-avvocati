## Key Findings

### Critical factual nuance — pre vs post 1 Sept 2024 regime

The Agenzia Entrate page (the authoritative source) confirms:

**Pre-1-Sept-2024 violations** keep the **old reduction ladder** with sanction base **30%**:
- 1/10 → ≤ 30 gg (= 1,5% del 15%)
- 1/9 → ≤ 90 gg (= 1,67% del 15%)
- 1/8 → entro dichiarazione (= 3,75% del 30%)
- 1/7 → entro dichiarazione anno successivo (= 4,29% del 30%)
- 1/6 → oltre dichiarazione anno successivo / oltre 2 anni (= 5% del 30%)
- 1/5 → dopo PVC

**Post-1-Sept-2024 violations** apply the **new reduction ladder** with sanction base **25%**:
- 1/10 → ≤ 30 gg (= 1,25% del 12,5%)
- 1/9 → ≤ 90 gg (= 1,39% del 12,5%)
- 1/8 → entro dichiarazione / 1 anno (= 3,125% del 25%)
- 1/7 → **oltre dichiarazione / oltre 1 anno** — UNIFIED tier (= 3,57% del 25%)
- 1/6 → dopo **schema d'atto** non preceduto da PVC (procedural milestone, not temporal)
- 1/5 → dopo PVC senza adesione e prima dello schema d'atto
- 1/4 → dopo schema d'atto post-PVC senza istanza adesione

**⚠️ Outdated content trap**: Studio Ripa and CafInforma both still show a "Lunghissimo / oltre 2 anni / 1/6 / 4,17%" tier for the post-reform regime. This is **incorrect**. Under D.Lgs. 87/2024 the simple temporal ladder ends at 1/7 = oltre 1 anno; the 1/6 reduction now requires the schema d'atto procedural milestone. Our page must get this right.

### Tasso legale storico (from Agenzia Entrate, authoritative)

- 2019: 0,8% | 2020: 0,05% | 2021: 0,01% | 2022: 1,25% | 2023: 5% | 2024: 2,5% | 2025: 2% | **2026: 1,60%**
- Note: D.M. 10 dicembre 2025 sets the 2026 rate
- Calcolo formula: `Imposta × tasso legale × giorni / 36500` (Agenzia Entrate official formula)
- For ravvedimenti a cavallo d'anno → split-period proration: each day uses the rate in force on that day (e.g., 16 giorni nel 2026 → 1,60%; resto del periodo 2025 → 2,00%)

### Dwell-Time Tool Recommendation

**Build an interactive multi-imposta calculator** with the following features:
1. Input: imposta dovuta (€), data scadenza, data ravvedimento, tipo tributo (dropdown)
2. Output: sanzione ridotta, interessi pro-rata, totale F24
3. Auto-detect pre/post 1-Sept-2024 regime based on data scadenza
4. Auto-detect ravvedimento tier (sprint/breve/intermedio/lungo/ultrannuale)
5. Pro-rata interest split for periods crossing year boundaries (e.g., 2025→2026)
6. Auto-suggest codici tributo F24 (imposta + sanzione + interesse) based on selected tributo
7. Generate a "Prospetto F24" summary that users can copy/print

This is a major UX upgrade over Andreani's iframe-based form (which only handles imposte dirette) and beats Studio Polli's calculator on transparency by showing the breakdown of the calculation step-by-step.
